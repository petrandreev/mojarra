We pass a literal string as the value of the "item" attribute in the composite component below. This literal string must be set into the value of the corresponding javaBeans property on the java top level component for javaTopLevelComponent.

Also, we do not pass an "action" attribute. The composite component has a default value for the "action" attribute, which is javaTopLevelComponent2. This means that pressing the login button will cause navigation to javaTopLevelComponent2.
