JavaServerTM Faces
Sample Applications

Specification Version: 1.2 Public Final Draft (PFD) Release
Reference Implementation (RI) Version: 1.2

This release of JavaServer Faces technology includes these examples.
For building the examples, see https://javaserverfaces.dev.java.net/faq.html#Code_build.
For deploying the examples,

  • If you are using Glassfish, drop the war file to GLASSFISH.HOME/domains/domain1/autodeploy
  • If you are using any other container, refer to the container's documentation.

    cardemo
    An online car store, localized in American English, French, German and Latin American Spanish. This application is a proof-of-concept application, which demonstrates that JavaServer Faces technology can be used to build industrial-strength Web applications. It is not intended to be representative of best practices for Web application design. To run the sample, visit http://:/jsf-cardemo
    webtier sample
    A simple application demonstrating JavaServer Faces technology inter-operating JSTL, JSP. To run the sampple, visit http://:/webtier-sample
    components
    A set of custom components. This application shows how to build various custom components with JavaServer Faces technology. To run the sample, visit http://:/jsf-components
    guessNumber
    Duke is thinking of a number. Can you guess it? This simple application shows a small set of standard components, page navigation, and validation. To run the sample, visit http://:/jsf-guessNumber
    non-jsp
    A simple application without JSP pages. This application demonstrates that you can build JavaServer Faces applications that don't use JSP pages. To run the sampple, visit http://:/jsf-nonjsp
    renderkits
    Demonstrates how to create and use a custom render kit and shows how easy it is to utilize multiple render kits in one application. To run the sampple, visit http://:/jsf-renderkits
