JavaServerTM Faces
1.0 EA1 Release
Documentation Bundle

JavaServer Faces is a toolkit for creating server-side graphic user interfaces for web applications. It contains a custom JSP tag library for building JSF applications that render to an HTML client, and an API and reference implementation that provide tag handling, object management, rendering, and event processing.
