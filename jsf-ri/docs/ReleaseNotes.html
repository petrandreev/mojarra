What This Release Includes

     
  • html_basic.tld: An custom tag library for representing components
  • javax.faces: An API for representing component state and handling events
  • A reference implementation that implements tag handling, object management, rendering, and event processing.
  • JavaServer Faces, a tutorial to help you get started.

Current Limitations

The purpose of this release is to introduce you to the JavaServer Faces framework and to provide you with an opportunity to give us feedback on the work we've done so far. This release has several limitations.
