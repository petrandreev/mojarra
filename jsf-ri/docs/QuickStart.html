Quick Start Guide

Here is a brief guide for getting yourself set up to use JavaServer Faces. Refer to the tutorial for more information on using JavaServer Faces.

  • After downloading the web services pack, set these environment variables:

    JAVA_HOME = the location of your JDK installation.
    JWSDP_HOME = the location of your web services pack installation.

    For more information on setting up your environment, see the Web Services Pack Release Notes and the Preface of the Web Services Pack tutorial.

  • Add this tag library directive to your JSP pages to reference the JSF tag library:

    <%@ taglib uri="http://java.sun.com/j2ee/html_basic/" prefix="faces" %>

  • See the JavaServer Faces tutorial for help with using the tag library.

  • Create a web application deployment descriptor, called web.xml, to reference FacesServlet and SessionListener. See the example web.xml file in the docs/examples/web/helloDuke directory for help with writing the web.xml file. Copy this file to your WEB-INF directory.

  • Add the jsf-api and jsf-ri jar files in {$JWSP_HOME}/tools/jsfaces to your classpath when you compile your classes, or copy them into the WEB-INF/lib directory of your web application before you build it.
